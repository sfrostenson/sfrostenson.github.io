Midterm Elections 2014


Working closely with Anthony DeBarros I helped architect the data sources feeding our elections forecast tool, which included poll averages from Real Clear Politics, campaign finance data from the Sunlight Foundation's Realtime Influence Explorer API and biographical information from the Project Vote Smart API.

In addition to updating the data with a series of Python scripts run by cron jobs outputting static json files from a Postgres database, we also used a Django admin to organize reporting and predictions from the USAT politics team.

Explore the forecast tool and results tool (which like newsrooms everywhere used AP data to show live results).
