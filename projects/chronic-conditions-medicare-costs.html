Building a Geolocator App for County-level Data on Chronic Conditions


Using Medicare data from 2012, I worked with Meghan Hoyer to visualize the health care cost of seniors with multiple chronic conditions. The project utilizes a Backbone router to set custom views of the interactive based on state and county views and uses D3 for the national view and Mapbox for the state views with D3 visualizations for county level data.

Additionally, I used an IP lookup to detect user's location on desktop and HTML5 Geolocation for mobile. Read the full story and explore the interactive.
